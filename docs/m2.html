이런 걸 해봐요

- Claude Code 설치, 로그인 - 실습 github repo 내려받기 (zip) - Obsidian 설치 및 열기 - 크롬, 옵시디언 확장프로그램 설치 (웹 클리퍼, 커뮤니티 플러그인 등)

핵심 내용

내 컴퓨터에 직접 환경 세팅
사용 도구

각 도구의 역할을 확인합니다

도구맡는 일
Claude Code자료를 읽고 정리해 답하기
Obsidian정리된 결과를 그래프로 보여 주기
웹 클리퍼웹에서 본 글을 폴더에 담기
실습 폴더스킬·예제·점검 도구 보관
터미널은 쓰지 않습니다. Node.js도 필요 없습니다.
1단계

Claude Code 설치하고 로그인

  1. 운영체제에 맞는 설치 파일을 내려받습니다

    맥은 dmg, 윈도우는 exe. 용량이 큽니다.

  2. 설치 후 배포받은 계정으로 로그인합니다
  3. 가운데 위의 Code 탭을 누릅니다

    파일을 직접 다루는 것은 Code 탭뿐입니다.

요금제를 올리라는 안내가 나오면 계정이 잘못된 것입니다. 손을 들어 주세요.
윈도우만 해당

Git 설치하기

맥은 이미 들어 있습니다. 윈도우만 git-scm.com 에서 받아 설치합니다. 설치 중 선택지는 전부 기본값으로 둡니다.

필요한 이유

윈도우에는 맥·리눅스가 기본으로 갖춘 파일 처리 도구가 없습니다. Git을 설치하면 그 도구가 함께 들어옵니다.

사용 여부

설치만 하면 됩니다. 오늘 git 명령을 입력할 일은 없습니다.

2단계

실습 폴더 내려받기

스킬, 예제 자료 두 개, 점검 도구가 들어 있습니다. 직접 만들 것은 없습니다.

  1. 저장소 주소로 들어갑니다

    github.com/yjh9929/okf-wiki-starter

  2. 초록색 Code 버튼 → Download ZIP

    파일 하나가 내려받아집니다.

  3. 찾기 쉬운 곳에 압축을 풉니다

    바탕화면이나 문서 폴더에 풉니다. 폴더 이름은 my-wiki 로 바꿔 둡니다.

폴더 안에 raw/ 와 wiki/ 가 있습니다. raw에는 예제 두 개, wiki는 비어 있습니다.
3단계

Code 탭에서 그 폴더를 엽니다

4단계

Obsidian 설치하고 같은 폴더 열기

연 다음 왼쪽의 그래프 아이콘을 눌러 그래프 뷰를 켭니다. 지금 비어 있는 것이 정상입니다.
5단계 · 가장 놓치기 쉬운 곳

웹 클리퍼 설치하고 저장 위치 지정

6단계 · 여유가 되면

옵시디언 커뮤니티 플러그인

오늘 실습에 필요하지는 않습니다. 앞 단계가 끝난 분만 진행합니다. 설정 → 커뮤니티 플러그인 → 제한 모드 해제 → 탐색에서 검색 → 설치.

플러그인무엇이 좋아지나
Dataview표식으로 노트를 한 번에 모아 봅니다. type이 같은 문서를 전부 목록으로 뽑을 수 있습니다.
Templater새 노트에 표식을 자동으로 넣어 줍니다. 맨 앞 몇 줄을 손으로 칠 필요가 없어집니다.
Omnisearch노트가 많아졌을 때 더 똑똑하게 검색합니다. PDF 안까지 찾습니다.
확인

여기까지 되면 준비 끝입니다

  • Claude Code에서 실습 폴더가 열려 있다
  • Obsidian에 같은 폴더가 열려 있고, 그래프 뷰가 켜져 있다
  • 브라우저에 웹 클리퍼 아이콘이 보인다
  • 웹 클리퍼 저장 위치가 raw 로 지정되어 있다
네 개가 다 되면 손을 들어 주세요. 먼저 끝난 분은 Code 탭에 "raw 폴더 읽어서 위키로 정리해 줘" 를 미리 넣어 봅니다.
막혔을 때

대부분 이 넷 중 하나입니다

증상이렇게 해보세요
Code 탭이 안 눌림로그인한 계정이 맞는지 확인하세요. 요금제 안내가 뜨면 계정 문제입니다.
폴더 선택이 안 됨 (윈도우)Git이 설치되어 있는지 확인하세요. 윈도우는 Git이 있어야 내 컴퓨터의 파일을 다룰 수 있습니다.
폴더를 못 찾겠음압축을 어디에 풀었는지 다시 확인하세요. 내려받기 폴더에 그대로 있는 경우가 많습니다.
그래프에 아무것도 없음아직 정리를 안 돌렸으면 정상입니다. 다음 모듈에서 채웁니다.
해결되지 않으면 손을 들어 주세요. 혼자 오래 붙잡고 있지 않아도 됩니다.
