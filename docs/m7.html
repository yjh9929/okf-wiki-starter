오래 쓰는 방법

수업 뒤에는 이 셋을 반복합니다

  1. 좋은 글을 보면 클리핑

    웹 클리퍼 버튼 한 번이면 raw/에 쌓입니다.

  2. 가끔 정리 실행

    "raw 읽어서 위키로 정리해 줘" — 쌓인 자료가 개념으로 묶입니다.

  3. 막히면 내 위키에 질문

    "내 위키 근거로 설명해 줘" — 출처 달린 답이 다시 쌓입니다.

한 달에 한 번 "위키 점검해 줘"를 실행하면 그래프가 무너지지 않습니다.
