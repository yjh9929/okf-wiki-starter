# 📚 나만의 AI 지식 관리 시스템 만들기 (okf-wiki-starter)

> **연세대학교 학부생 대상 4시간 집중 특강 실습용 Starter Repository**  
> *"질문할 때마다 다시 뒤지지 말고, 수집할 때 미리 정리하자!" (RAG → LLM Wiki)*

📑 **세미나 교안 슬라이드 →** <https://yjh9929.github.io/okf-wiki-starter/>

---

## 💡 강의 개요 (Overview)

매번 LLM에게 질문할 때마다 긴 문서를 다시 검색(RAG)하고 계신가요?  
이 워크숍에서는 **Obsidian**과 **Claude Code**, 그리고 **OKF(Obsidian Knowledge Format)**를 활용하여 **'공부하고 수집할수록 스스로 연결되고 축적되는 나만의 지식 위키'**를 구축하는 방법을 배웁니다.

* 🏢 **Obsidian**: 나만의 생각과 메모가 담기는 **작업실**
* 🧑‍💼 **AI (Claude)**: 자료를 요약하고 정돈해 주는 **사서**
* 📖 **LLM Wiki**: 쌓일수록 지식의 복리 효과를 내는 **나만의 서재**

---

## 🎯 학습 목표 (Key Objectives)

1. **RAG의 한계와 LLM Wiki 이해**: 매번 검색하는 RAG 대신 사전 정리(Pre-indexing)가 더 효율적인 이유를 이해합니다.
2. **AI 실습 환경 구축**: Claude Code CLI와 Obsidian, Chrome Web Clipper를 내 컴퓨터에 직접 세팅합니다.
3. **지식 축적의 선순환 체득**: 수집한 자료를 위키 문서로 합성하고, 질문을 통해 정제된 답변을 다시 위키로 저장하는 복리 구조를 경험합니다.
4. **지식 그래프 시각화**: Obsidian Graph View를 읽는 법을 익히고, 고립된 노트(Orphan) 및 누락된 링크를 점검합니다.

## 📂 디렉토리 구조 (Directory Structure)

본 저장소를 내려받으면 아래와 같은 폴더 구조로 구성되어 있습니다.

```text
okf-wiki-starter/
├── 📁 raw/            # 1. 웹 클리퍼나 파일로 수집한 원본 자료(.md, .txt)를 넣는 공간
├── 📁 wiki/           # 2. AI가 OKF 규칙에 맞춰 요약·합성한 개념 위키 노트가 생성되는 공간
├── 📁 prompts/        # 3. AI 에이전트 실행에 필요한 OKF 프롬프트 및 스킬 정의 파일
├── 📁 .obsidian/      # 4. 옵시디언 커뮤니티 플러그인 및 그래프 뷰 설정 파일
└── 📄 README.md       # 강의 실습 안내서 (현재 문서)
```

## ❓ FAQ

* **Q. 관심 주제나 전공 관련 수집할 자료가 떠오르지 않으면 어떻게 하나요?**  
  - 수업 현장에서 제공하는 예시 주제 리스트(요리, 기업 신기술 동향 등) 중 하나를 선택해 시작하시면 됩니다.
* **Q. 코딩 경험이 전혀 없는데 따라갈 수 있나요?**  
  - 네! 직접 코드를 짜지 않고 AI 명령어(프롬프트)와 템플릿을 활용하므로 누구나 끝까지 완성할 수 있습니다.
