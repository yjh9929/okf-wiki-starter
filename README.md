# 🧠 나만의 AI 지식 허브 만들기 — OKF 지식 위키 실습 템플릿

> Claude Code × Obsidian으로, 흩어진 자료를 **AI가 스스로 정리·연결한 나만의 지식 위키**로 만드는 실습 템플릿입니다.
> 연세대학교 × 모두의연구소 세미나용 · Karpathy의 [LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) 패턴 + 구글 OKF 기반.

**이 repo를 내려받아 바로 시작하세요.** 상단의 **"Use this template"** 버튼을 쓰거나, Claude Code에게 이렇게 부탁하면 됩니다:
> "이 repo를 `my-wiki` 폴더로 clone 해줘: `https://github.com/kisdevan/okf-wiki-starter`"

**무언가를 직접 만들 필요는 없습니다.** 정리·질문·점검을 해 주는 도구(`okf-wiki` 스킬)가 이미 들어 있으니, 여러분은 **개념을 이해하고 직접 써보는 데**만 집중하시면 됩니다.

👉 **처음이라면 [세팅_가이드.md](세팅_가이드.md) 를 먼저 보세요** (6단계, 10분).

---

## 📁 이 폴더에 뭐가 들어 있나요?

```
├── .claude/skills/okf-wiki/   ← 정리·질문·점검을 해 주는 AI 스킬 (이미 설치됨)
├── raw/                       ← 원본 자료를 넣는 곳 (예제 2개가 이미 있어요)
├── wiki/                      ← AI가 정리한 결과가 쌓이는 곳 (지금은 비어 있음)
├── lint_okf.py               ← 위키가 잘 됐는지 검사하는 도구
└── README.md                 ← 지금 보고 있는 이 파일
```

---

## 🚀 시작하기

처음이라면 **[세팅_가이드.md](세팅_가이드.md)** 를 먼저 보세요. 6단계로 차근차근 안내합니다:
Claude Code 설치·로그인 → 이 repo 내려받기 → Obsidian 설치 → **웹 클리퍼 확장(필수)** → **추천 플러그인 3개(Dataview·Templater·Omnisearch)** → 확인.

**이미 세팅을 마쳤다면**, 아래 3가지만 기억하면 됩니다:
- **Obsidian**: 이 폴더를 "보관함(Vault)"으로 열고 그래프 뷰 켜기
- **터미널**: 이 폴더에서 `claude` 실행 → `/okf-wiki` 뜨면 준비 끝
- **웹 자료**: 브라우저 웹 클리퍼로 좋은 글을 `raw/`에 저장 → 위키로 정리

### 실습 시작 — 이렇게 말만 하면 됩니다

| 하고 싶은 것 | 이렇게 말하세요 |
|---|---|
| 📥 **정리 (Ingest)** | `/okf-wiki` 또는 "raw 폴더 읽어서 위키로 정리해 줘" |
| ❓ **질문 (Query)** | "위키 내용 근거로 ○○ 알려줘. 출처도 달고, 쓸만하면 저장해 줘" |
| ✅ **점검 (Lint)** | "위키 점검해 줘" 또는 아래 명령 |

```bash
PYTHONIOENCODING=utf-8 python3 lint_okf.py wiki
```

- **Obsidian 그래프**에서 개념들이 선으로 연결되는 걸 확인하세요.
- `wiki/index.md`(목차), `wiki/log.md`(작업 기록), 그리고 개념 문서들이 생겼는지 보세요.
- 검사에서 "ALL PASS ✅"가 나오면 완성입니다! (끊긴 링크 0, 외톨이 문서 0)

## 💡 내 주제로 해보기
`raw/`의 예제 대신 **내가 관심 있는 자료**(기사·논문·메모)를 넣어 보세요.
텍스트를 복사해 `raw/내주제.txt`로 저장하고 다시 "정리해 줘"라고 하면, 나만의 위키가 만들어집니다.

## 🆘 잘 안 될 때
| 증상 | 이렇게 해보세요 |
|---|---|
| `/okf-wiki`가 안 보여요 | Claude Code 버전 확인(2.1.145+), 이 폴더 안에서 `claude`를 실행했는지 확인 |
| 한글이 깨져요 (Windows) | 명령 앞에 `PYTHONIOENCODING=utf-8` 붙이기 |
| 그래프에 선이 안 보여요 | 문서에 `[[...]]` 연결이 있는지, Obsidian이 이 폴더를 열었는지 확인 |

즐겁게 만들어 보세요. 궁금한 건 언제든 조교에게 물어보세요! 🙌
