# 🛠️ 시작하기 — 환경 세팅 가이드 (처음 한 번만!)

세미나를 편하게 따라오려면 아래 세팅을 미리 해두세요.
어렵지 않아요. **차근차근 6단계**만 따라오면 됩니다. 막히면 언제든 조교에게 손! 🙋

> 💡 **가능하면 수업 1~2일 전에 1~5단계까지 마쳐 오세요.** 설치가 사람마다 시간이 제일 많이 걸리는 부분이라, 미리 해두면 수업 시간을 실습에 온전히 쓸 수 있어요.

---

## 1단계 · Claude Code 설치하고 로그인하기
내 자료를 정리해 줄 "AI 사서"입니다.

- **설치** (터미널에 붙여넣기)
  - macOS / Linux: `curl -fsSL https://claude.ai/install.sh | bash`
  - Windows (PowerShell): `irm https://claude.ai/install.ps1 | iex`
- **로그인**: 터미널에 `claude` 입력 → 브라우저가 열리면 본인 Claude 계정으로 로그인
  - ※ `claude auth login` 같은 명령은 없어요. 그냥 `claude`만 치면 됩니다.
- **확인**: `claude --version` 이 버전 숫자를 보여주면 성공 ✅

> 💳 본인 Claude 구독(Pro 이상)으로 동작하며, **API 추가 요금은 붙지 않습니다.**

---

## 2단계 · 제공된 실습 repo 내려받기 (Claude Code에게 시키기)
직접 git 명령을 외울 필요 없어요. Claude Code에게 **말로** 부탁하면 됩니다.

1. 터미널에서 `claude` 실행
2. 이렇게 입력:
   > "이 깃허브 repo를 `my-wiki` 폴더로 clone 해줘: `<수업에서 안내하는 repo 주소>`"
3. Claude Code가 알아서 `git clone`을 실행해 줍니다. (권한 물어보면 승인)

- 📌 **Windows 사용자**: git이 없으면 clone이 안 돼요. [Git for Windows](https://git-scm.com/download/win)를 먼저 설치하세요. (macOS는 대부분 이미 설치됨)
- 내려받은 `my-wiki` 폴더 안에는 **AI 스킬(okf-wiki) · 예제 자료 · 점검 도구**가 이미 들어 있습니다.

---

## 3단계 · Obsidian 설치하고 폴더 열기
정리된 지식을 **그래프**로 보여주는 프로그램입니다.

1. https://obsidian.md 에서 설치
2. Obsidian 실행 → **"Open folder as vault(폴더를 보관함으로 열기)"** → 2단계에서 받은 `my-wiki` 폴더 선택
3. 왼쪽 도구모음의 **그래프 아이콘**을 눌러 그래프 뷰를 켜 두세요 (아직은 비어 있어요)

> ✅ **그래프 뷰**와 **Properties(속성, YAML 편집 UI)** 는 Obsidian에 **기본 내장**이라 따로 설치할 게 없어요.

---

## 4단계 · [필수] 웹 클리퍼 확장 설치 (웹 자료를 바로 담기)
관심 있는 웹 기사·문서를 **버튼 한 번으로 `raw/` 폴더에 저장**하는 공식 브라우저 확장입니다. 자료 수집이 정말 편해져요.

- **설치** (내 브라우저에 맞게 하나)
  - Chrome/Edge/Brave/Arc: [Chrome 웹스토어 — Obsidian Web Clipper](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf)
  - Firefox: [Firefox 애드온](https://addons.mozilla.org/en-US/firefox/addon/web-clipper-obsidian/)
  - Safari: [App Store](https://apps.apple.com/us/app/obsidian-web-clipper/id6720708363)
  - Edge 애드온: [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/obsidian-web-clipper/eigdjhmgnaaeaonimdklocfekkaanfme)
- **간단 설정**: 확장 아이콘 클릭 → 설정에서
  - **기본 보관함(Vault)** = `my-wiki` 선택
  - **저장 위치(Note location)** = `raw` 로 지정
  - (선택) 템플릿에 속성 `type: source` 를 미리 넣어두면, 저장하는 순간 OKF 표식이 자동으로 붙어요.
- 이제 웹에서 좋은 자료를 보면 확장 버튼만 눌러 `raw/`에 쏙 담고, 수업에서 위키로 정리하면 됩니다.

## 5단계 · [추천] 옵시디언 커뮤니티 플러그인 3개
있으면 훨씬 편한 도구들이에요. Obsidian **안에서** 설치합니다.

**설치 방법** (공통):
`설정(⚙️)` → `커뮤니티 플러그인` → **"커뮤니티 플러그인 켜기"**(제한 모드 해제) → `탐색(Browse)` → 이름 검색 → `설치` → `사용(Enable)`

| 플러그인 | 왜 좋은가 (초보자 관점) |
|---|---|
| **Dataview** | `type` 표식으로 노트를 한 번에 모아 보기. 예: "type이 person인 노트 전부 목록" — OKF 위키에 **제일 유용**해요. |
| **Templater** | 새 노트에 OKF 프론트매터(`type:` 등)를 **한 번에 자동 삽입**. YAML을 손으로 안 쳐도 됩니다. |
| **Omnisearch** | 위키가 커졌을 때 **더 똑똑한 검색**(PDF 내용까지). 노트가 많아지면 켜세요. (선택) |

> 딱 3개면 충분해요. Graph Analysis·Juggl·Metadata Menu 같은 건 초보 단계에선 **설치하지 마세요**(복잡하거나 관리가 중단됨). 나중에 익숙해지면 그때 둘러보면 됩니다.
>
> ✨ 참고: 최신 Obsidian에는 코드 없이 표/카드로 노트를 모아 보는 **Bases**(기본 코어 기능)도 있어요. Dataview 문법이 부담되면 `설정 → 코어 플러그인 → Bases`를 켜 보세요. (버전에 따라 없을 수 있음)

## 6단계 · 잘 됐는지 확인 (30초 체크)
- [ ] 터미널에서 `my-wiki` 폴더에 들어가 `claude` 실행 → `/okf-wiki` 가 뜬다
- [ ] Obsidian에서 `my-wiki`가 열리고 그래프 뷰가 보인다
- [ ] 브라우저에 Obsidian Web Clipper 아이콘이 있다
- [ ] Dataview·Templater 가 "사용 중"이다

모두 체크됐다면 준비 끝! 이제 수업에서 신나게 만들어 봐요. 🎉

## 🌱 수업 후에도 계속 쓰는 팁
- **자료 모으기**: 좋은 웹 글을 보면 웹 클리퍼로 `raw/`에 저장 → 나중에 `/okf-wiki`로 위키에 편입
- **질문하기**: 공부하다 막히면 `claude`에게 "내 위키 근거로 설명해줘"라고 물어보기
- **꾸준히 점검**: 가끔 "위키 점검해줘"로 끊긴 링크·빈틈 정리 → 위키가 **쓸수록 똑똑해집니다**
- **어디서든**: 위키는 그냥 마크다운 폴더라, 나중에 다른 컴퓨터·다른 도구로 옮겨도 그대로 열립니다.
